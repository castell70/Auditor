        AuditorMonitor - Guía de Uso

        ¿Qué es AuditorMonitor?

        AuditorMonitor es una herramienta integral para la planificación, seguimiento y gestión de auditorías. Permite organizar participantes, definir etapas de auditoría, programar actividades y visualizar el cronograma mediante un diagrama de Gantt.

        Secciones Principales

        1. Auditoría

        Gestiona la información general de la auditoría. Crea un código único, asigna un nombre descriptivo y selecciona al auditor líder. Puedes descargar tus datos en formato JSON o cargar un archivo JSON existente para recuperar proyectos anteriores.

        2. Participantes

        Registra a todos los miembros del equipo de auditoría y personal involucrado. Asigna roles específicos (Auditor líder, Auditor, Responsable de proceso, Observador) y gestiona el directorio de contactos con correos electrónicos. Puedes editar o eliminar participantes en cualquier momento.

        3. Etapas

        Define las fases principales de la auditoría (planeación, ejecución, informe, etc.). Para cada etapa asigna un nivel de riesgo (bajo, medio, alto), agrega actividades específicas, carga archivos adjuntos y establece fechas de inicio y cierre.

        4. Calendario

        Visualiza y programa todas las etapas y actividades en un formato de calendario. Ajusta las fechas de inicio y fin para cada etapa y actividad de manera intuitiva. El sistema permite ver claramente la distribución temporal del proyecto.

        5. Diagrama

        Observa un diagrama de Gantt visual que muestra la línea de tiempo completa del proyecto. Las barras naranja representan etapas y las verdes representan actividades. Puedes descargar un informe completo en PDF que incluya todos los detalles de la auditoría.

        Flujo de Trabajo Recomendado

        1. Crear auditoría: Ingresa un código y nombre en la sección "Auditoría"
        2. Registrar participantes: Agrega el equipo de trabajo en "Participantes"
        3. Definir etapas: Crea las fases principales en "Etapas" con sus actividades
        4. Programar cronograma: Asigna fechas en "Calendario"
        5. Visualizar y reportar: Revisa el diagrama en "Diagrama" y descarga el informe en PDF

        Características Clave

        • Gestor de roles personalizable: Define roles específicos para tu organización
        • Guardado automático: Los cambios se guardan en tiempo real en la sesión
        • Exportación de datos: Descarga tus proyectos como JSON para respaldos o compartir
        • Importación de datos: Carga proyectos guardados previamente
        • Datos de ejemplo: Carga datos de muestra para familiarizarte con la herramienta
        • Diagrama de Gantt: Visualización temporal de todas las actividades
        • Generación de reportes: Crea reportes en PDF con toda la información de la auditoría

        Consejos Útiles

        • Usa códigos de auditoría consistentes para identificar fácilmente tus proyectos
        • Asigna siempre un auditor líder para claridad en la responsabilidad
        • Define actividades claras y específicas dentro de cada etapa
        • Mantén actualizado el cronograma para un seguimiento preciso del proyecto
        • Descarga regularmente respaldos de tus datos en JSON
        • Utiliza niveles de riesgo para priorizar y enfatizar áreas críticas
